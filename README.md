# CodeAlpha_FAQChatbot

## Project Overview

This project is an AI-powered FAQ Chatbot developed using Python, NLTK, Scikit-learn, and Tkinter GUI.

The chatbot answers frequently asked questions by matching the user's query with the most similar FAQ using Natural Language Processing (NLP) techniques and Cosine Similarity.

## Features

* FAQ-based chatbot
* NLP preprocessing using NLTK
* TF-IDF Vectorization
* Cosine Similarity Matching
* Interactive GUI using Tkinter
* User-friendly interface

## Technologies Used

* Python
* Tkinter
* NLTK
* Scikit-learn

## Installation

Install required libraries:

pip install nltk scikit-learn

Run the application:

python faq_chatbot.py
